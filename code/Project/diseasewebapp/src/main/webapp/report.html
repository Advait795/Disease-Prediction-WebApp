Note:
P(Disease|Symptoms) : The chance of having the disease given that you're showing specific symptoms.
P(Disease) : The chance of having the disease without considering any symptoms.
P(Symptoms|Disease) : The chance of showing specific symptoms if you already have the disease.

Naive Bayes Bias/Assumption :
Naive Bayes assumes that all features are independent of each other given the class label, allowing the model to calculate the joint probability as the product of individual probabilities.

Note : The probabilities are derived from historical records for each disease, with the following sample sizes:

      Hypertension: 26,083 records
      Stroke: 40,910 records
      Diabetes: 100,000 records
